Teclas de usuario


Cuando el algoritmo no se está ejecutándo:


  • Simple click: añadir punto / mover punto ( si algun punto ha sido seleccionado anteriormente).

  • Long click: seleccionar punto.

  • ← Borrar punto ( si algun punto ha sido seleccionado anteriormente).


Cuando el algoritmo se está ejecutando:

  • Simple click: Si método paso a paso → siguiente paso.

  • Doble click : Cambio a método paso a paso.

  • ↑ cambio a método directo.

  • ↓ cambio a método delay.

Los cambios en la forma de realizar los algoritmos tambien se pueden realizar mediante el menu.
